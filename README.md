# Api

## Project setup

```
run 'npm install' and then run 'npm install dotenv' because sometimes this module refuse to install with the others.
```

## Database setup

```
Create an SQL database using your database management software, then execute the files that are into database folder
in order to create the db tables.

When everything is ok, create a file named .env into the root folder of this project. Into this new file you need to insert the following lines:

HOST = <your database host. If runing in localhost then write localhost or 127.0.0.1>
USER = <your database connection user>
PASSWORD = <your database connection password>
DATABASE = <your database name>
```

## Run the Api
```
To run the application just navigate into the root folder in your command prompt and run 'npm start'
```
